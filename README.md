# Logic Circuit

서원대학교 2학년 1학기 **논리회로** 학습 자료와 LogicWorks 실습 파일을 정리한 저장소입니다.  
논리 게이트, 불 대수, 조합논리회로, 순서논리회로, 메모리와 프로그래머블 논리 장치를 단원별로 관리합니다.

---

## ✨ 핵심 내용

| 분류 | 내용 |
| --- | --- |
| 이론 | 논리 게이트, 불 대수, 조합논리회로, 플립플롭, 레지스터, 메모리 |
| 실습 도구 | LogicWorks 5 |
| 자료 구성 | 단원별 PDF, 과제, 회로 파일, 발표 자료 |
| 팀 과제 | Hamming Code, BCD to 7-Segment, 4bit to 32bit 회로 |

---

## 📂 저장소 구조

| 경로 | 설명 |
| --- | --- |
| `1장/` ~ `12장/` | 단원별 강의자료와 과제 |
| `LogicWorks 5/` | LogicWorks 설치 파일과 사용법 자료 |
| `로직웍스 메뉴얼ppt 과제/` | LogicWorks 사용자 메뉴얼 발표 자료 |
| `팀 과제/` | 팀 프로젝트 회로 파일과 발표 자료 |

## ▶️ 사용 방법

`.cct` 파일은 LogicWorks 5에서 열어 회로를 확인합니다.

```text
LogicWorks 5 -> File -> Open -> *.cct
```

PDF와 HWP/PPT 자료는 강의 복습과 과제 제출 기록 확인용으로 정리되어 있습니다.
